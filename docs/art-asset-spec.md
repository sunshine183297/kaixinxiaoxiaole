# 猫猫消消乐 - 美术素材需求规格书

## 整体风格要求

参考图片风格：**日系卡通/可爱萌系**
- 猫咪头像为主视觉，正面或略微侧面
- 大眼睛、圆润轮廓、柔和色调
- 粉色系/暖色系背景基调
- 线条清晰、填充柔和渐变
- **无水印、无Logo、无文字**
- 统一美术风格，所有素材保持一致的绘制质感

## 六、素材放入项目后的操作步骤

### 棋子素材
1. 将 7 张猫咪头像 PNG 放入 `assets/Texture/Cells/cats/`
2. 在 Cocos Creator 中打开项目，编辑器会自动生成 `.meta` 文件
3. 使用 TexturePacker 或 Creator 内置工具打包为 plist 图集（可选）
4. 编辑每个 Prefab（`assets/Prefabs/Bear.prefab` → 替换为橘猫, `Cat.prefab` → 替换为英短蓝猫 ...），将 Sprite 的 SpriteFrame 指向新素材
5. 验证动画帧引用正确

### 对应关系
```
Bear.prefab   → 橘猫 (orange_tabby.png)     CELL_TYPE.A
Cat.prefab    → 英短蓝猫 (british_shorthair.png)  CELL_TYPE.B
Fox.prefab    → 布偶猫 (ragdoll.png)          CELL_TYPE.C
Chicken.prefab→ 暹罗猫 (siamese.png)          CELL_TYPE.D
Frog.prefab   → 三花猫 (calico.png)           CELL_TYPE.E
Horse.prefab  → 黑猫 (black.png)              CELL_TYPE.F
Bird.prefab   → 金色喵星人 (golden_special.png)  CELL_TYPE.BIRD
```

### 障碍物素材
1. 将 3 张障碍物 PNG 放入 `assets/resources/obstacles/`
2. 代码会通过 `ResourceLoader.loadSpriteFrame('obstacles/ice')` 自动加载

### UI 素材
1. 直接替换 `assets/resources/level-select/` 下的同名文件
2. 直接替换 `assets/Texture/login/` 和 `assets/Texture/ui/` 下的同名文件

## 七、AI 绘图批量生成建议

### 推荐工具
- **豆包 AI**: 已有参考风格图，直接上传参考图 + 提示词
- **Midjourney**: 使用 `--style raw --ar 1:1 --no text watermark logo`
- **Stable Diffusion**: 使用 `masterpiece, best quality` + 负向提示 `text, watermark, logo`

### 批量生成流程
1. 先生成 1 张橘猫作为风格锚点（用你提供的参考图做 img2img 或风格参考）
2. 用同一种子/风格参数生成其他 6 只猫
3. 统一后处理：裁剪为正方形、去背景、调整到 200×200
4. 障碍物和 UI 图标用同一风格参数生成

### 统一提示词前缀（建议）
```
(masterpiece, best quality), cute kawaii anime style, soft pastel colors, 
pink theme, clean lines, simple background, game asset icon, 
no text, no watermark, no logo, transparent background PNG,
```
