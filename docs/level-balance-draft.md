# 猫猫消消乐 - 关卡系统设计文档

## 1) 系统架构

猫猫消消乐采用**混合关卡系统**：手工设计关卡 + 程序化无限生成。

```
关卡 1-20:   手工设计（levels.json）
关卡 21+:    程序化生成（LevelGenerator）
```

### 核心文件

| 文件 | 职责 |
|------|------|
| `assets/resources/levels.json` | 手工关卡数据（1-20关） |
| `assets/Script/Model/Level/LevelGenerator.ts` | 程序化关卡生成器 |
| `assets/Script/Model/Level/LevelConfig.ts` | 关卡配置服务（混合加载） |
| `assets/Script/Controller/LevelSelectController.ts` | 关卡选择（分页无限滚动） |

## 3) 程序化生成系统（第 21+ 关）

### 设计原则

1. **确定性随机**：同一 levelId 永远生成同一关卡（SeededRandom）
2. **平滑难度曲线**：难度系数 `1 - 1/(1 + (id-20)*0.02)` 趋近于 1
3. **内容多样性**：12 种棋盘模板、70+ 种关卡名、3 种障碍物、收集目标
4. **渐进解锁**：每 3 星解锁下一关，保证玩家始终有目标

### 难度参数随 levelId 的变化

| 参数 | 低 (21关) | 中 (50关) | 高 (100+关) |
|------|---------|---------|-----------|
| 难度系数 | 0.02 | 0.37 | 0.62 |
| 步数 | ~28 步 | ~22 步 | ~16 步 |
| 时间 | ~118 秒 | ~100 秒 | ~80 秒 |
| 目标分数 | ~450 | ~1350 | ~2000 |
| 棋盘形状 | 3种简单 | 7种中等 | 12种全部 |
| 障碍物 | 无 | ice + chain | ice + chain + crate |
| 收集目标 | 无 | 偶尔出现 | 经常出现(1-2种猫咪) |

### 棋盘形状模板（12种）

| # | 名称 | 说明 | 首次出现 |
|---|------|------|---------|
| 1 | 完整方形 | 标准 9×9 | 第 21 关 |
| 2 | 圆角方形 | 四角切去 | 第 21 关 |
| 3 | 菱形 | 旋转45°的菱形 | 第 21 关 |
| 4 | 猫脸 | 猫耳朵 + 圆脸轮廓 | 第 31 关 |
| 5 | 鱼骨 | 鱼骨架形状 | 第 31 关 |
| 6 | 爪印 | 三个爪垫 + 掌心 | 第 31 关 |
| 7 | 十字 | 十字形 | 第 31 关 |
| 8 | H形 | H字母形状 | 第 51 关 |
| 9 | 大圆 | 更大的圆形 | 第 51 关 |
| 10 | 双翼 | 翅膀形状 | 第 51 关 |
| 11 | 蝴蝶结 | X形蝴蝶结 | 第 51 关 |
| 12 | 框形 | 嵌套矩形框 | 第 51 关 |

### 障碍物引入节奏

| 关卡范围 | 障碍物 | 说明 |
|---------|--------|------|
| 21-29 | 无 | 纯消除，熟悉程序化关卡 |
| 30-39 | 冰冻鱼骨 (ice) | 1-2 层冰覆盖 |
| 40-54 | + 毛线球 (chain) | 锁定格子需消除解锁 |
| 55+ | + 纸箱子 (crate) | 阻挡下落的障碍 |

### 收集目标

- 第 35 关起可能出现
- 约 60% 的关卡有收集目标
- 1-2 种猫咪品种，每种需收集 5-28 个
- 品种名：橘猫、蓝猫、布偶、暹罗、三花、黑猫

### 解锁规则

```
第 21 关解锁需要: 59 星 (56 + 3×1)
第 22 关解锁需要: 62 星 (56 + 3×2)
第 N 关解锁需要: 56 + 3×(N-20) 星
```

### 星级评价公式

```
3星: score >= target × 2.1
2星: score >= target × 1.5
1星: score >= target
```

## 5) 关卡选择页分页机制

- 首次加载：显示手工关卡（1-20关）
- 如果全部解锁，自动加载下一页（每页 20 关）
- 滚动到底部自动触发加载更多
- 遇到未解锁关卡时停止继续加载
- 无上限，玩家可以持续解锁新关卡

## 6) 后续扩展方向

4. **社交排行榜**：微信好友分数排名
5. **关卡编辑器**：允许玩家创建和分享关卡
