# Copilot Instructions for 猫猫消消乐

## 项目概述

猫猫消消乐是一款以猫咪为主题的三消游戏，使用 Cocos Creator 3.8.6 + TypeScript 开发，目标平台为微信小游戏。

## 项目结构

- `assets/Script/Controller/` - 控制器（LoginController, LevelSelectController, GameController）
- `assets/Script/Model/` - 数据模型和游戏逻辑（GameModel, CellModel, LevelConfig 等）
- `assets/Script/View/` - 视图组件（GridView, CellView, EffectLayer）
- `assets/Script/Utils/` - 工具类（WechatCompat, ThemeConfig, TextConst, AudioUtils 等）
- `assets/resources/` - 运行时加载资源（关卡 JSON、UI 图标）
- `build-templates/wechatgame/` - 微信小游戏构建模板

## Cocos Creator 3.x 注意事项

- 使用 TypeScript，开启严格类型检查
- Cocos Creator 版本：3.8.6
- 使用 `cc` 模块导入引擎 API
- 微信小游戏兼容使用 `WechatCompat` 工具类

## 微信小游戏相关

- 微信 API 调用统一通过 `WechatCompat` 类
- 所有平台特定代码都做了安全降级处理
- AppID 配置在 `build-templates/wechatgame/project.config.json`
